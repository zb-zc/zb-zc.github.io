    • 构造函数的目的：创建一个对象

    • 构造函数的名字要跟类的名称相同

    把构造函数放到private里面

    这样操作的作用：不允许外界创建对象；比如单例模式

    常量成员函数 （const）

    在函数后面加const ,如下所示： 

    double real () const { return re; }
double imag () const { return im; }

    class 类里面分为会改变数据的和不会改变数据的函数，只要是不会改变数据的直接加上const，比如上面的函数，只是为了返回re的值，不会去改变re的值，所以加上了const
    创建一个对象如下

    complex c1(2,1);
cout << c1.real();

    但是如果没有加const，下面这种情况编译就会出现问题 

    const complex c1(2,1);
cout << c1.real();

    创建对象的时候声明 c1的参数是不可以改变的，但是在complex类中如果没有加const就说明 re是可以修改的。所以编译的时候就会报错

    const reference 和 non-const reference

    const reference

    const refernence 只可读不可写，可以绑定不同类型的const/非const对象，或者不可寻址的值。如字面值常量。const引用只是表明，保证不会通过此引用改变被引用的对象，但是还可以通过其他方式来改变。编译器产生临时变量的时候引用必须为const

    non-const reference

    非const引用可读也可写，必须绑定相同类型的非const对象。
    实例如下：

    右值引用

    左值（可在等号左边出现的元素）：具名对象、在栈数据段和堆数据段上分配的对象、其他对象的数据成员，或者一切具有确定存储范围的数据项。
    右值:字面值和临时变量。
    左值引用只可以绑定左值，不可以绑定右值。但是右值可以绑定到const 左值引用上。

    右值引用:只与右值绑定的引用。通过 && 进行申明。

    参数传递

    • pass by value:传递所有数据
    • pass by reference(引用):在底层就相当于传递指针 “&”。这种速度比较快
      （尽量传引用）
    • return by value :返回数据
    • return by reference: 在可以的情况下尽量用引用的方式返回
      什么情况下不可以 return by reference
      在函数结束后，变量被释放，这种情况下不可以将此变量的引用传递出去，除了这种情况，其他情况都可以返回引用。

    操作符重载

    在c++中操作符就是一种函数。

    成员函数 this

    所有的成员函数都带有this指针，指向成员函数属于的对象

    操作符重载（类成员函数）

    两个操作数的 例如“+=”

    • 规则： 左边的是调用者，右边是被调用者，如下

        inline complex&
      __doapl (complex* ths, const complex& r)
      {
      ths->re += r.re;
      ths->im += r.im;
      return *ths;
      }
  inline complex&
      complex::operator += (const complex& r)
      {
      return __doapl (this, r);
      }
  
  complex c1;
  complex c2;
  c2 += c1 //c2 = c2 + c1;

    如上：this 指的是c2（左值），r指的是c1

    • 传送者无需知道接收者是以reference形式接受，比如

      c2 += c1
      在下面重载函数中 c1对应的重载函数中的形参，但是形参的类型是reference，c1是一个value。这就是所说的传送者无需接收者是以reference形式接受。
      还有一种情况是函数返回值无需知道函数返回类型是reference。如下 

        inline complex&
          __doapl (complex* ths, const complex& r)
          {
          ths->re += r.re;
          ths->im += r.im;
          return *ths;
          }

    操作符重载 （非成员函数）

    inline complex
operator + (const complex& x, const complex& y)
{
return complex (real (x) + real (y), imag (x) + imag (y));
}

inline complex
operator + (const complex& x, double y)
{ 
return complex (real (x) + y, imag (x));
}

inline complex
operator + (double x, const complex& y)
{
return complex (x + real (y), imag (y));
}
    • 以上的函数不可return by reference; 返回一个对象，对象在函数执行完毕之后分配的临时空间被释放。
    • typename + ():创建临时对象。

    拷贝赋值

    操作符重载“=”，还是同一类对象。这种操作就叫做拷贝赋值。如下：

    String& operator=(const String& str); 

    String& String::operator=(const String& str)
{
if (this == &str)
    return *this;

delete[] m_data;
m_data = new char[ strlen(str.m_data) + 1 ];
strcpy(m_data, str.m_data);
return *this;
} 
complex s1("hello");
complex s2(s1);//这里用到了拷贝构造
s2 = s1;//这里用到了拷贝赋值
因为“=”操作符重载，等号左边是操作对象等号右边是被操作的对象。所以操作符重载中this 是等号左边的对象。在拷贝赋值中分以下爱几个步骤：
1. 删除自身数据。
2. 创建足够大的空间
3. 进行拷贝操作。
注意在起手就要进行检测自我赋值：浅拷贝之后不能进行自身的拷贝赋值。因为操作的是同一片地址空间，然而拷贝赋值第一步是delete。

    lambda函数

    lambda函数：若某表达式需要一个函数，则可以等到所需之处才进行定义 。

    lambda匿名函数的定义

    1. [外部变量访问方式说明符]：不能省略
      [ ] 方括号用于向编译器表明当前是一个 lambda 表达式，其不能被省略。在方括号内部，可以注明当前 lambda 函数的函数体中可以使用哪些“外部变量”

    2. (参数) ：可以省略
      和普通函数的定义一样，lambda 匿名函数也可以接收外部传递的多个参数。和普通函数不同的是，如果不需要传递参数，可以连同 () 小括号一起省略

    3. mutable：可以省略
      此关键字可以省略，如果使用则之前的 () 小括号将不能省略（参数个数可以为 0）。默认情况下，对于以值传递方式引入的外部变量，不允许在 lambda 表达式内部修改它们的值（可以理解为这部分变量都是 const 常量）。而如果想修改它们，就必须使用 mutable 关键字。
      注意，对于以值传递方式引入的外部变量，lambda 表达式修改的是拷贝的那一份，并不会修改真正的外部变量

    4. noexcept/throw() ：可以省略
      可以省略，如果使用，在之前的 () 小括号将不能省略（参数个数可以为 0）。默认情况下，lambda 函数的函数体中可以抛出任何类型的异常。而标注 noexcept 关键字，则表示函数体内不会抛出任何异常；使用 throw() 可以指定 lambda 函数内部可以抛出的异常类型。

    5. -> 返回值类型 ：可以省略
      指明 lambda 匿名函数的返回值类型。值得一提的是，如果 lambda 函数体内只有一个 return 语句，或者该函数返回 void，则编译器可以自行推断出返回值类型，此情况下可以直接省略-> 返回值类型

    6. 函数体 ：不可以省略
      和普通函数一样，lambda 匿名函数包含的内部代码都放置在函数体中。该函数体内除了可以使用指定传递进来的参数之外，还可以使用指定的外部变量以及全局范围内的所有全局变量
      需要注意的是，外部变量会受到以值传递还是以引用传递方式引入的影响，而全局变量则不会。换句话说，在 lambda 表达式内可以使用任意一个全局变量，必要时还可以直接修改它们的值。
